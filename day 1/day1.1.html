Ex: console.error('an error occured'); [3]console.warn() used to log warning message to the console, highlighted in yellow color. Ex: console.warn('Warning!'); [4]console.clear() used to clear the console. Ex: console.clear(); [5]console.time() and console.timeEnd() used to indicate the time spent by a block of code, function,etc. it only needs the same label for time() and timeEnd() method. Ex: for(let i=0;i<5;i++) { console.count(i); } [8]console.group and console.groupEnd() used to group contents in a seperate block. it uses the same label in the group() and groupEnd() method.
