[1]STRING METHODS ->charCodeAt() Return the Unicode of the last character in a string. Ex : var str = "HELLO WORLD"; var n = str.charCodeAt(str.length-1); ->concat() Joins two or more strings, and returns a new joined strings. Ex : var str1 = "Hello "; var str2 = "world!"; var res = str1.concat(str2); -> fromCharCode() Converts Unicode values to characters. Ex : var res = String.fromCharCode(72, 69, 76, 76, 79); ->indexOf Checks whether a string contains the specified string/characters Ex : var str = "Hello world, welcome to the universe."; var n = str.includes("world"); ->match() Searches a string for a match against a regular expression, and returns the matches. Ex : var str = "The rain in SPAIN stays mainly in the plain"; var res = str.match(/ain/g); ->substring() Extracts the characters from a string, between two specified indices. Ex : var str = "Hello world!"; var res = str.substring(1, 4); ->split() Splits a string into an array of substrings. Ex : var str = "How are you doing today?"; var res = str.split(" "); ->slice() Extracts a part of a string and returns a new string. Ex : var str = "Hello world!"; var res = str.slice(0, 5); ->replace() Searches a string for a specified value, or a regular expression, and returns a new string where the specified values are replaced. Ex : var str = "Visit Microsoft!"; var res = str.replace("Microsoft", "W3Schools"); [2]ARRAY METHODS ->Converting arrays to strings The JavaScript method toString() converts an array to a string of (comma separated) array values. Ex : var fruits = ["Banana", "Orange", "Apple", "Mango"]; document.getElementById("demo").innerHTML = fruits.toString(); ->Changing elements Ex : var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits[0] = "Kiwi"; // Changes the first element of fruits to "Kiwi" ->Deleting Elements Since JavaScript arrays are objects, elements can be deleted by using the JavaScript operator delete. Ex : var fruits = ["Banana", "Orange", "Apple", "Mango"]; delete fruits[0]; // Changes the first element in fruits to undefined ->Splicing an Array The splice() method can be used to add new items to an array. Ex : var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.splice(2, 0, "Lemon", "Kiwi"); ->Using splice() to Remove Elements With clever parameter setting, you can use splice() to remove elements without leaving "holes" in the array. Ex : var fruits = ["Banana", "Orange", "Apple", "Mango"]; fruits.splice(0, 1); // Removes the first element of fruits ->Slicing an Array The slice() method slices out a piece of an array into a new array. This example slices out a part of an array starting from array element 1 ("Orange") Ex : var fruits = ["Banana", "Orange", "Lemon", "Apple", "Mango"]; var citrus = fruits.slice(1);
